📄️ Access Sage sensors

A Sage sensor is an entity that produces measurements of a phenomenon and that helps users analyze what is happening in the environment. There are sensors already hosted by Sage and also sensors that are being integrated into Sage as a user-hosted sensor. A sensor does not necessarily mean a physical device, but can be a program producing measurements from data -- we call it software-defined sensor. Once those sensors become available in Sage nodes edge applications running inside the nodes can pull measurements from the sensors to process them.
